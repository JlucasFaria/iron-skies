=== IronSkies - Instruções de Entrega ===

1. COMPILAR O EXECUTAVEL
   Execute o arquivo build.bat (duplo clique ou pelo terminal).
   O executavel sera gerado em: dist\IronSkies.exe

2. MONTAR A PASTA DE ENTREGA
   dist\
   ├── IronSkies.exe
   └── assets\
       ├── images\
       └── sounds\

   Copie a pasta assets\ para dentro de dist\ manualmente.

3. GERAR O ZIP
   Compacte a pasta dist\ inteira em IronSkies.zip

4. REQUISITOS ATENDIDOS
   [x] Jogo 2D jogavel (nao e console/cmd)
   [x] Menu principal com controles exibidos na tela
   [x] Codigo original
   [x] Compilado para Windows (.exe)
   [x] Funciona sem assets (usa formas geometricas como fallback)
