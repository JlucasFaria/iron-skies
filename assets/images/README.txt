Sprites opcionais do jogo IronSkies.
Se ausentes, o jogo usa formas geométricas como fallback.

Arquivos esperados (PNG com fundo transparente):
  player.png     — sprite do avião do jogador
  enemy.png      — sprite do avião inimigo
  background.png — imagem de fundo (800x600)

Use assets/loader.py para carregar: carregar_imagem(caminho, largura, altura)
